- Next.js App Router

- shadcn-style component patterns

## Ownership Rules

This repository owns:

- landing-page information architecture
- section copy and CTA structure
- Next.js and Tailwind implementation details

## Working Rules

- Keep section IDs, CTA IDs, and approved content structure stable unless the task explicitly changes them.
- Use the shared ParkTek palette and token semantics from `$parktek-platform`.
- Keep landing-specific composition, density, and storytelling decisions local to this repo.
