# ParkTek Landing

Next.js marketing site for ParkTek.

## Quick Start

```bash
npm install
npm run dev
```

Run the site at `http://localhost:4000`.

## Common Commands

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`

## Local Docs

- `docs/image-requirements.md`: content asset constraints
- `docs/landing-implementation.md`: section structure, UI direction, and content ownership rules

## Shared Standards

Use `$parktek-platform` for shared ParkTek design tokens and cross-platform standards. This repository owns landing-page information architecture, copy implementation, and Next.js-specific UI behavior.
